# bbnet-scripts
A repo holding all scripts that run BanBansNetwork

# setup

```
sudo adduser banbans
su banbans
cd
mkdir bbnet
cd bbnet
git clone https://github.com/BanBansNet/bbnet-scripts
cd bbnet-scripts
./setup.sh
```

# administration

```
#start server GER 1
./start.sh 1
#start server GER 2
./start.sh 2
#stop server GER 2
./stop.sh 2
#stop all servers
./stop.sh -1
```
